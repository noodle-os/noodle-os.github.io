NoodleOS

A lightweight and fast Linux-based CLI operating system for programmers and hobbyists.

NoodleOS is designed to be ultra-minimal, using only 5MB of storage and 11MB of RAM, making it perfect for older hardware or embedded systems. It features a custom package manager called noodle, designed to handle package installation and updates efficiently in a minimal environment.

Install

NoodleOS comes preinstalled, so there's no need for a setup process. Simply download the image and you're ready to go!

For installation on your own hardware, follow the steps:

  • Download the NoodleOS image from the Download Page.
  • Write the image to a USB drive or SD card using a tool like Rufus or Balena Etcher.
  • Boot your system from the USB/SD card to start using NoodleOS.
